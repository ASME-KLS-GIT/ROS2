# Welcome to the C++ track. 
---
This track includes writing the features like nodes , services etc. in C++ . This track is going to be more setup intensive , but it will however lead to a better understanding of the ROS2 ecosystem. 

The python track is much less difficult when compared to the C++ track , this track has a steeper learning curve , but overcompensates it by providing absolute control over what we can do with ROS , while providing a very fast speed of execution and an ability to implement advanced tasks.

Using this track would allow you to have a very uniform language palette since embedded code used in robots and micro-controllers is written in C.
